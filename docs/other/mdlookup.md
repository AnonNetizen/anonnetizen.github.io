# Markdown速查

行之间需加空行，要不然会连在一起

## eg.

```markdown
1.

#

## 

...//后面+空格加标题，n个#n级

2.

3.
    ```语言//Markdown对语言名的支持很强大，比如Markdown既可以写成Markdown又可以写成md，大小写不敏感
    //此处不用空行
    代码块
    //此处也不用空行
    ```

4.

`行内语句`

>引用 //引用可嵌套

6.

//下面三个都是HTML标识

<sub>下标</sub>

<sup>上标</sup>

<center>文字</center> //居中

7.

--- //分割线
