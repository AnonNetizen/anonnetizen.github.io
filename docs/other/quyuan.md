# 屈原生平

## 简介

屈原（约公元前340—公元前278年），战国时期楚国诗人、政治家。

>屈原者，名平，楚之同姓也。

>皇览揆余初度兮，肇锡余以嘉名。名余曰正则兮，字余曰灵均。——《离骚》

屈原是楚国贵族，芈姓屈氏，名平字原，楚武王熊通之子屈瑕的后代。

## 仕途前期

周慎靓王元年（公元前320年），屈原应楚怀王之召出山进京，在鄂渚为县丞，这是屈原仕途的开始。

>为楚怀王左徒。

周慎靓王二年，屈原升任楚怀王左徒。

屈原人生前期楚怀王在位。至于“左徒”到底是干什么的多有争议。

### 楚怀王

*……卒以此见怀王之终不悟也。*

>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;人君无愚智贤不肖，莫不欲求忠以自为，举贤以自佐。然亡国破家相随属，而圣君治国累世而不见者，其所谓忠者不忠，而所谓贤者不贤也。怀王以不知忠臣之分，故内惑于郑袖，外欺于张仪，疏屈平而信上官大夫、令尹子兰，兵挫地削，亡其六郡，身客死于秦，为天下笑，此不知人之祸也。

```
早期，楚怀王任用贤能，楚国一度人才济济，国势大盛。楚怀王积极变法，但改革触动了权贵阶层的利益，遭到了他们的抵抗，改革失败 。北伐魏国，攻城夺地；东败越国，拓境江东；甚至组织起七国联军，西伐强秦。但中后期误信张仪，毁坏齐、楚联盟，先后败于秦、齐，失去汉中等地。误用佞臣子椒、子兰、靳尚、上官大夫，宠爱郑袖，排斥屈原，致使国事日非。
```

>上官大夫与之同列，争宠而心害其能。怀王使屈原造为宪令，屈平属草稿未定。上官大夫见而欲夺之，屈平不与，因谗之曰：“王使屈平为令，众莫不知。每一令出，平伐其功，曰以为‘非我莫能为也。’”王怒而疏屈平。

周赧王元年（公元前314年），屈原因楚王听信上官大夫谗言而被罢黜左徒之官，任三闾大夫之职。

## “放逐乃赋离骚”

周赧王二年，屈原被流放汉北。

>屈平疾王听之不聪也，谗谄之蔽明也，邪曲之害公也，方正之不容也，故忧愁幽思而作《离骚》。

被流放后，屈原感到痛心，忧愁楚国前景，写下了历史评价极高的《离骚》。

>十三年，庶长章击楚於丹阳，虏其将屈匄，斩首八万；又攻楚汉中，取地六百里，置汉中郡。——《史记·秦本纪》

屈原被流放后张仪破楚齐联盟，汉中郡沦陷，史称“丹阳之战”。

## 流放南方

>其后，诸侯共击楚，大破之，杀其将唐眜。时秦昭王与楚婚，欲与怀王会。怀王欲行，屈平曰：“秦，虎狼之国，不可信，不如毋行。”怀王稚子子兰劝王行：“奈何绝秦欢！”怀王卒行。入武关，秦伏兵绝其后，因留怀王，以求割地。怀王怒，不听。

>长子顷襄王立，以其弟子兰为令尹。

周赧王十六年，屈原此时已从汉北流放地返回，力劝怀王不要赴会，可怀王的幼子子兰竭力怂恿怀王前去。结果怀王一入武关就被秦军扣留，劫往咸阳，要挟他割让巫郡和黔中郡，楚怀王没有同意。此后楚国内顷襄王继位。

>亡走赵，赵不内。复之秦，竟死于秦而归葬。

周赧王十九年，楚怀王死于秦国。

>楚人既咎子兰以劝怀王入秦而不反也。屈平既嫉之，虽放流，眷顾楚国，系心怀王，不忘欲反……令尹子兰闻之，大怒。卒使上官大夫短屈原于顷襄王。顷襄王怒而迁之。

同年，屈原遭到第二次放逐，到了江南，洞庭湖、长江流域。屈原此时开始广泛接触群众。

*啊，我思念那洞庭湖，我思念那长江……——《雷电颂》*

周赧王二十一年到周赧王三十六年，屈原被流放到南方的荒僻地区。时间长达十六年，这段时期屈原写下了大量优秀的作品，如《九章·悲回风》等。

## 见渔父，投汨罗

>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;屈原既放，游于江潭，行吟泽畔，颜色憔悴，形容枯槁。渔父见而问之曰：“子非三闾大夫与？何故至于斯？”屈原曰：“举世皆浊我独清，众人皆醉我独醒，是以见放。”  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;渔父曰：“圣人不凝滞于物，而能与世推移。世人皆浊，何不淈其泥而扬其波？众人皆醉，何不餔其糟而歠其醨？何故深思高举，自令放为？”  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;屈原曰：“吾闻之，新沐者必弹冠，新浴者必振衣；安能以身之察察，受物之汶汶者乎？宁赴湘流，葬于江鱼之腹中。安能以皓皓之白，而蒙世俗之尘埃乎?”  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;渔父莞尔而笑，鼓枻而去，乃歌曰：“沧浪之水清兮，可以濯吾缨；沧浪之水浊兮，可以濯吾足。”遂去，不复与言。&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;——《渔父》

周赧王三十七年，秦昭襄王诈以公主许配给楚顷襄王，屈原长跪城外力谏不果。秦军趁顷襄王开城迎亲，长驱直进，攻入楚都郢。屈原投汨罗江自尽。

## 大事年表

周显王二十九年（前340）正月初七日，一说周显王三十年（前339）正月十四日，屈原生于楚国丹阳秭归。

周慎靓王元年（前320），在鄂渚为县丞，仕途开始。

周慎靓王二年（前319），升任楚怀王左徒。

周赧王元年（前314），被罢黜左徒之职，任三闾大夫。

周赧王二年（前313），被流放汉北。

周赧王十九年（前296），被放逐到江南。

周赧王二十一年（前294）~周赧王三十六年（前279），被流放到南方的荒僻地区。

周赧王三十七年（前278），投汨罗江自尽，时年62岁左右。
